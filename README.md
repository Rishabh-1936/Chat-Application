# Chat-Application
A simple Chat Application developed using node.js and mongodb

## **Steps to use Chat Application:**
1.Install node.js 

2.Install the mongodb driver

3.Install express.js ,mongodb,mongoose,socket.io.,socketio-auth:

     Write command in terminal:
  
     npm install express mongodb mongoose socket.io socketio-auth --save
  
     This will install all the required packages and library.

4.Start the server:

    node server.js

5.Start the mongodb server

6.Go to the link localhost:4000
